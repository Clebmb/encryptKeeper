# encryptKeeper 0.1.0

## Current Features

- Open a folder vault containing `.gpg` note files
- Create, open, edit, rename, and delete encrypted notes
- Import public and private OpenPGP keys
- Select a private key for decryption
- Select recipient keys for encryption
- Unlock and lock the session with auto-lock support
- Configure the auto-lock timer or disable it
- Optional auto-save while unlocked
- Compact desktop UI for notes, keys, and session controls

## MSI or NSIS?

Choose `MSI` if:

- You want a standard Windows installer
- You may deploy through IT/admin tools
- You prefer a more conventional install/uninstall flow

Choose `NSIS` if:

- You want a lighter, simpler installer
- You are installing it yourself on a single PC
- You just want the quickest normal setup experience

## Quick Recommendation

- For most individual users: use `NSIS`
- For managed or enterprise-style Windows environments: use `MSI`
